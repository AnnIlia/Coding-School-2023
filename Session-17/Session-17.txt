Απαντήσεις του Session-17:


1. εάν αυτό το Request που θα κάνουμε είναι από τον Server και όχι από τον Client.


2. θα πρέπει να είναι : image/jpeg


3. ισως με το να ορίσουμε ένα header σχετικά με αυτο. 
Για παράδειγμα: εάν το ορίσουμε ως "Cash-Control", θα πρέπει να αναφέρουμε ως maximum τα δευτερόλεπτα στο οποίο θέλουμε να διατηρηθούν.
Στη περίπτωση αυτή η μέρα είναι 86400 δευτερόλεπτα, οπότε θα εισάγουμε τον αριθμό των δευτερολέπτων.


4. ναι. Απλά η εφαρμογή με μόνο αυτά τα αιτήματα δεν θα μπορεί να τροποποιεί δεδομένα στον Server, 
θα δίνεται στον Client μόνο η επιλογή να ανακτά πληροφορίες.


5. ναι. Απλά η με μόνο αυτά τα αιτήματα δεν θα ανακτούσε πληροφορίες από τον Server,
ο Client θα μπορούσε μόνο να στείλει δεδομένα σε αυτόν. Ίσως για αποθήκευση;


6. το status code σε αυτή τη περίπτωση θα είναι το: 400 Bad Request.
Εμφανίζεται όταν λείπει κάποιο από τα υποχρεωτικά πεδία. Για παράδειγμα΄(αυτό που υπόθηκε και στη παράδοση):
όταν σε ένα Login παραλείπεται το username ή το password.


7. το status code σε αυτή τη περίπτωση θα είναι το: 500 Internal Server Error.
Εμφανίζεται όταν κάτι δεν έπαιξε στον Server. Τα γνωστό σε όλους "Oops.." ή "Sorry..".


8. ίσως όχι. Το Get χρησιμοποιείται όταν θέλουμε ως Clients να πάρουμε μία πληροφορία *χωρίς* να αλλάξουμε το resource.
Το αίτημα Post χρησιμοποιείται για αυτή τη περίπτωση, καθώς δηλώνει πως θέλουμε να κάνουμε δημιουργία resource πίσω στον Server.
Ίσως με τη χρήση ενός header "Cookie", το οποίο όμως θα πρέπει να υπάρχει στο Cookie Storage του browser που χρησιμοποιούμε, 
να μπορεί να γίνεται και μόνο με το Get. 


9. Το αίτημα Post χρησιμοποιείται για τη δημιουργία resource πίσω στον Server, 
ενώ το αίτημα Put για να κάνει ενημέρωση ένα resource πίσω σε αυτόν.


10. Το αίτημα Post χρησιμοποιείται για τη δημιουργία resource, οπότε θα μπορούσε να χρησιμοποιηθεί γι' αυτόν τον σκοπό.
Μαζί με κάποιο header, το οποίο θα όριζε τον τύπο των αρχείων (Mime Format) που είναι "ευπρόσδεκτα" στον Server 
εάν θέλουμε κάποια συγκεκριμένα formats ή με ως "multipart/form-data" εάν αυτά δεν είναι συγκεκριμένα.
